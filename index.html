Mini Minecraft（still working on the project）

Used Qt and OpenGL to make a Mini Minecraft game. Player can move around in fly mode and ground mode. And blocks can be created and destroyed by player. The terrain is procedual generated, and thus has no boundary. This project is the final project of the course "Interactive Computer Graphics" in 2021 fall, and we are still adding new features. Half-edge Mesh and Skeleton Editor

Used Qt and OpenGL to make a mesh and skeleton editor. It can load obj model file and json skeleton file to edit them. Implemented operation include ray casting selection, add midpoint, triangulate, subdivision, two methods of skinning, and two methods of blending.
